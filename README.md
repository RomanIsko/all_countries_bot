### Development
* Create `.env` file with `BOT_TOKEN` variable
  ```sh
  BOT_TOKEN=11111111:AAAAAAAAAAAAAAAAAAAAAAAAAAA
  ```
* Install NodeJS dependencies
  ```sh
  npm install
  ```  
* Then run npm `dev` script
  ```sh
  npm dev
  ```
